## 🎨 推奨形式

### スクリーンショット
- 形式: PNG (透明背景対応)
- 解像度: 1920x1080以上
- DPI: 144dpi (Retina対応)

### アイコン
- 形式: SVG (ベクター) + PNG (ラスター)
- カラー: ブランドカラー (#267332)
- 背景: 透明

### サンプル画像
- 形式: PNG/JPG
- サイズ: 最大1MB以下
- 説明テキスト付き
